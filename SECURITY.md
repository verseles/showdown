# Security Policy

## Supported Versions

We actively maintain the following versions of Showdown.Best with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Showdown.Best seriously. If you believe you have found a security vulnerability, please report it to us as described below.

### How to Report

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them by email to: security@showdown.best

### What to Include

When reporting a vulnerability, please include:

1. **Description**: A clear description of the vulnerability
2. **Steps to Reproduce**: Detailed steps to reproduce the issue
3. **Expected vs Actual Behavior**: What you expected to happen vs what actually happened
4. **Impact**: The potential impact of this vulnerability
5. **Environment**: Browser/OS version, device type, etc.

### Response Timeline

- **Initial Response**: We will acknowledge receipt of your vulnerability report within 48 hours
- **Investigation**: We will investigate and confirm the vulnerability within 5 business days
- **Resolution**: We will work to resolve confirmed vulnerabilities within 30 days

### Security Best Practices

When using Showdown.Best, please follow these security best practices:

1. **Keep your browser updated**: Use the latest version of your browser
2. **Verify data sources**: Always verify benchmark data from official sources
3. **No sensitive data**: This application does not require or store sensitive personal information
4. **Use HTTPS**: Always access the site over HTTPS in production

## Recognition

We believe in giving credit where credit is due. If you report a security vulnerability that leads to a fix, we will:

- Acknowledge your contribution in our security advisories
- Include your name in our Hall of Fame (unless you prefer to remain anonymous)
- Provide appropriate recognition for your responsible disclosure

## Scope

This security policy applies to:

- The main Showdown.Best website
- All public repositories under the showdown-best organization
- All official Showdown.Best applications and services

## Out of Scope

The following are not covered by this security policy:

- Third-party websites linked from Showdown.Best
- User-generated content or comments
- Issues in browser extensions or plugins

## Questions?

If you have questions about this security policy, please contact us at security@showdown.best
